# expose-to-the-light-web
expose-to-the-light-web client

## Install application
### Download installer
`curl -L -o install-web.sh https://www.dropbox.com/s/l4k3duj2naes8eq/install-web.sh?dl=1`

### Install
* `chmod u+x install-web.sh`
* `./install-web.sh`

### On Mac Os
* `brew install imagemagick`
* `brew install ufraw`
* `brew install exiftool`

#### On Raspberry Pi (unix)
* `sudo apt-get -y install openjdk-11-jdk`
* `sudo apt -y install imagemagick`
* `sudo apt -y install exiftool`

## Run the app
`ettl-web`
